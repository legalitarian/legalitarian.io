Accessibility Statement for Legalitarian Websites

This is an accessibility statement from Legalitarian LLC.

Measures to support accessibility

Legalitarian LLC takes the following measures to ensure accessibility of Legalitarian:

  • Test all web pages with assistive technology and web browsers on iOS, Android, MacOS, Windows, and Ubuntu Linux
  • Include people with disabilities in our user story personas
  • Include people with disabilities in user testing when possible

Conformance status

The Web Content Accessibility Guidelines (WCAG) defines requirements for designers and developers to improve accessibility for people with disabilities. It defines three levels of conformance: Level A, Level AA, and Level AAA. Legalitarian's webpages are fully conformant with WCAG 2.1 level AA. "Fully conformant" means that the content conforms to the accessibility standard without any exceptions.

Feedback

We welcome your feedback on the accessibility of Legalitarian web pages. Please let us know if you encounter accessibility barriers by emailing accessibility@legalitarian.io

We try to respond to feedback within 2 business days.

Technical specifications

Accessibility of Legalitarian pages relies on the following technologies to work with the particular combination of web browser and any assistive technologies or plugins installed on your computer:

  • HTML
  • CSS
  • JavaScript

These technologies are relied upon for conformance with the accessibility standards used.

Assessment approach

Legalitarian LLC assesses the accessibility of all of its web pages during development using Deque Systems' axe DevTools and Testing Library. Changes to pages must also pass manual testing using assistive technology in the latest Firefox, Chrome, Safari, and Edge on iOS, Android, MacOS, Windows, and Ubuntu Linux.
